What is an API?

An API is a program that helps you to do things in a simpler way; for example, Google Login is an API. It stands for Application Program Interface. It is really a set of routines, protocols, and tools to help programmers achieve a task. Back to Google Login, it allows programmers to easily access a user's personal information without the need to create a new program.
